ScriptSass - Introduction
The ScriptSass is a javascript Sass dynamic compiler. It works both with .sass files and inline scss code. Just load the file in a script tag using this line of javascript code -
ScriptSass.load("relative-url-to-file");
And, to load it from an inline code-block, write the code-block in a <code type="sass"></code> block, and run the following JS line-
ScriptSass.compileInline();
Please note that ScriptSass requires that you load JQuery first.
You can use this code to load JQuery followed ScriptSass :
	<script src="jquery-3.2.1.min.js"></script>
	<script src="jsass.current.js"></script>
A very basic syntax would be:
	$(document).ready(function (){
		ScriptSass.compileInline();  //compiles inline SCSS codes
	}

I am still working on this project. As of now, it supports only the following features:
  • SCSS syntax, not exactly Sass syntax. Semi-colons at end of statements are must
  • Sass variables (except Maps)
  • extend directive
  • mixins
  • nested css selectors
  • Code minification
Update

  • Removed support for @import to make some improvements to it
  • Added mixin support (may have gotchas)
  • Faster processing
